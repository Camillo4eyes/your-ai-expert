# 🤖 Your AI Expert

A simple Streamlit web app that lets you chat with different AI experts, each with their own personality and expertise. You can also choose the language of the conversation.

Built as a learning project to demonstrate basic LLM skills: conversation management, system prompts, streaming responses, and prompt engineering — without any complex architecture (no RAG, no vector DB, no fine-tuning).

---

## ✨ Features

- **6 AI Experts** to choose from, each with a unique personality and system prompt
- **6 Languages**: Italiano, English, Español, Français, Deutsch, Português
- **Streaming responses** — answers appear token by token, just like ChatGPT
- **Auto-reset** when you switch expert or language
- **Welcome message** generated by the expert themselves on first load
- Clean sidebar UI with expert descriptions

## 🚀 Setup

### 1. Clone the repository

```bash
git clone https://github.com/Camillo4eyes/your-ai-expert.git
cd your-ai-expert
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure your OpenAI API key

Copy the example file and add your key:

```bash
cp .env.example .env
```

Open `.env` and replace `your-api-key-here` with your actual [OpenAI API key](https://platform.openai.com/api-keys):

```
OPENAI_API_KEY=sk-...
```

### 5. Run the app

```bash
streamlit run app.py
```

The app will open automatically in your browser at `http://localhost:8501`.

## 🧠 How it works

### System prompts
Each expert is defined in `experts.py` with a name, emoji, description, and a detailed **system prompt** that shapes the AI's personality and expertise. The system prompt is injected at the beginning of every API call so the model always "stays in character".

### Language injection
The language chosen by the user (from `languages.py`) adds a short instruction to the system prompt — e.g., *"Always respond in English"* — which forces the expert to reply in the selected language regardless of what language the user writes in.

### Streaming
Responses are streamed using OpenAI's `stream=True` parameter and Streamlit's `st.write_stream()`, so text appears progressively instead of all at once.

### Session state
Streamlit's `st.session_state` keeps the full conversation history across reruns. When the user switches expert or language, the history is automatically cleared so the new expert starts fresh.
